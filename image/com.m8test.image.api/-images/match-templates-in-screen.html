matchTemplatesInScreen

abstract fun matchTemplatesInScreen(templatePath: String, similarity: Int?, limit: Int?, regionBuilder: Rect.() -> Unit?, outputPath: String?, method: Int?, parallel: Boolean?): Results<RegionResult>

从屏幕中查找 templatePath, 会自动申请屏幕截图权限

Return

匹配到的所有结果（按相似度降序），包含匹配区域和相似度

Parameters

templatePath

模板图像文件路径, 小图

similarity

匹配相似度，必须在 0, 100 范围内

regionBuilder

限定仅在原图像的该区域内查找

outputPath

匹配结果图像输出路径

method

模板匹配方法

parallel

是否启用并行处理

Throws

IllegalArgumentException

如果 similarity 无效或文件路径无效
