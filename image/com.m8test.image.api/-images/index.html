Images

interface Images : Variable

图片处理核心接口，提供屏幕截图录取证、图像裁剪、模板匹配、颜色识别等图像处理功能。

该接口封装了一系列图像处理操作，支持从文件或屏幕中进行图像分析， 包括模板匹配、颜色比较、特征点识别等高级功能，适用于自动化测试、图像识别等场景。

所有方法均提供异常处理机制，当参数无效或操作失败时会抛出相应异常， 建议调用时进行适当的异常捕获。
