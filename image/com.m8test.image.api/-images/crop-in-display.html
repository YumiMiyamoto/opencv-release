cropInDisplay

abstract fun cropInDisplay(displayId: Int, outputPath: String, regionBuilder: Rect.() -> Unit)

裁剪屏幕指定区域并保存为图像文件（自动申请权限）。

结合屏幕截图与区域裁剪功能，直接从指定屏幕截取矩形区域并保存。适用于快速获取 屏幕特定区域的图像，如控件截图、验证码区域等。

Receiver

待构建的矩形对象，通过设置left、top、right、bottom属性定义裁剪区域

Parameters

displayId

屏幕标识符：0表示物理屏幕镜像，其他值表示虚拟屏幕

outputPath

裁剪结果保存路径，若文件已存在会被覆盖

regionBuilder

用于定义裁剪区域的lambda表达式，坐标基于屏幕分辨率

Throws

SecurityException

当自动申请权限失败时抛出

IllegalArgumentException

当路径无效、屏幕ID不存在或区域设置不合法时抛出
