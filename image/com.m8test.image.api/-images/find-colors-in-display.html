findColorsInDisplay

abstract fun findColorsInDisplay(displayId: Int, firstColor: String, similarity: Int?, limit: Int?, config: FindColorsConfig.() -> Unit): Results<CoordinateResult>

在屏幕中查找匹配指定颜色的所有点（自动申请权限）。

扫描指定屏幕，返回所有与目标颜色相似的坐标点，按相似度降序排列。 适用于查找特定颜色的元素分布，如地图标记、数据可视化点等。

Return

包含匹配坐标的Results对象，每个结果包含坐标和相似度

Parameters

displayId

屏幕标识符：0表示物理屏幕镜像，其他值表示虚拟屏幕

firstColor

目标主颜色，支持RGB和ARGB格式

similarity

颜色相似度阈值（0-100），null使用默认值

limit

最大返回结果数量，null表示无限制

config

颜色查找配置构建器

See also

FindColorsConfig

了解更多配置选项

Throws

IllegalArgumentException

当参数无效或颜色格式错误时抛出

SecurityException

当自动申请权限失败时抛出
